Website

POPULUS様公式サイト

2025.05

URL

https://populus-design.com/

幼稚園・保育園向けのホームページ制作を行うPOPULUS様のWEBサイトを制作しました

  • 制作範囲

    企画 / 情報設計 / デザイン / コーディング

  • ターゲット

    幼稚園経営者（50代）、ホームページ制作に詳しくない方、金額にはシビアだが価値のあるものにはお金をかけたい方

  • 目的

    Web検索から公式LINEへ繋げる

  • 情報設計

    はじめに園のホームページについての悩みを記載し、悩みに共感したターゲットが自分ごととしてサイトを読み進めてもらえるようにしました。
    ホームページ制作を依頼するのが初めてで不安というターゲットに向けて、【料金パック】はパックに含まれる内容、追加料金がかかる内容を明確にしています。 また、【制作の流れ】についても園側で準備いただく部分をわかりやすく表記することで、ユーザーが安心して依頼できるようにしました。

  • デザイン

    ご覧になったユーザーが安心感を抱き、自然と内容に興味を持っていただけるよう、「優しさと分かりやすさ」をテーマにデザインを組み立てました。 サイト全体の基調となる色には心を落ち着かせる効果のある淡いグリーンを選び、フォントも角の取れたソフトでナチュラルな書体を採用することで、全体を通して柔らかく親しみやすい雰囲気を演出し、ユーザーが心理的なハードルを感じずに情報に触れられるよう工夫しています。
    パック内容も忙しいターゲットが一目で直感的に各プランのポイントを把握できるように、単に文字をリストアップするのではなく、 伝えたい情報を内容ごとにカードのように枠で区切りスッキリと整理して配置しました。

  • 制作期間

    企画 / 情報設計

    2週間

    デザイン

    3週間

    コーディング

    2週間

  • 使用ツール

    Figma / Visual Studio Code / Illustrator / Photoshop

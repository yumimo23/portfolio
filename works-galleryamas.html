Website

Gallery Amas様公式サイト

2024.12

URL https://gallery.echelle-totalbeauty.com/

デジタルハリウッドSTUDIOの卒業制作として、神奈川県平塚市のレンタルギャラリー Gallery Amas様の WEBサイトリニューアルを行いました

  • 制作範囲

    企画 / 情報設計 / デザイン / コーディング

  • ターゲット

    湘南地域在住で自分の作品や活動を発表したい男女

  • 課題

    趣味で作品を制作している方にも気軽に利用してもらいたいが、既存サイトはプロの芸術家向けで敷居の高い印象になっている

  • 目的

    ギャラリー利用の敷居を低くし、利用・見学問い合わせに繋げる

  • 情報設計

    ①既存サイトはギャラリーでの展示の様子が載っておらず利用用途が分かりづらかったため、利用事例のページを新設。 実際の展示写真を多く掲載することで、ギャラリーの雰囲気や利用・展示イメージが湧くようにしました ②ギャラリーの強みとして挙がった「はじめての方でも使いやすいギャラリー」をアピールするため、利用の流れや備品の案内を充実させ、 サイトを見たユーザーが「私にも簡単に展示ができるかも」と思ってもらえるようにしました

  • デザイン

    メインビジュアルはギャラリーのコンセプトである「人との繋がり」を円で表現しました。また、大きな円の縁に沿ってギャラリー内装などの 小さな円が浮き出てくるアニメーションをつけて一目で印象づけられるようにしています。
    既存サイトが黒白赤の配色だったのに対して、ギャラリーの白くて明るい内装の雰囲気が伝わる配色を意識しました

  • 制作期間

    企画 / 情報設計 / デザイン / コーディング

  • 使用ツール

    Figma / Visual Studio Code / Illustrator / Photoshop

  • 振り返りと今後の展望

    こちらの作品では初めてクライアント探しからサーバーアップまでの全行程を一人で行いました。
    情報設計にあたっては複数回のヒアリングにてクライアント様の強みや課題、ターゲットのすり合わせを行ったほか、ペルソナを設定しペルソナの悩みを解決できるようなコンテンツ提案も行いました。
    クライアント様の課題を解決する且つユーザーにとって必要な情報をサイト内に配置することに時間を要しましたが、サイトリニューアルの目的に沿って考えていくことでデザインや情報設計にブレがない制作ができたと考えます。
    スクールの担当講師からは「ユーザーのことを早い段階から客観視できていて、必要な情報の設計が上手くできた」と評価していただきました。

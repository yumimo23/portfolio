ABOUT

うえもと ゆみ

1989年に埼玉県行田市出身。現在は未就学児の息子を育てながらWEBデザイナーとして活動しています。

大学ではマーケティングを学び、その後は大手ベーカリーチェーンでの販売職、健診センターでの予約受付といったお客様と直接関わる仕事に約10年間従事。その後「サイト制作を通して課題を解決する」というWEBデザインの考え方に惹かれ、デジタルハリウッドSTUDIOでWEBデザインを学びました。

お客様対応業務の経験から「相手の目線に立った対応ができること」が私の強みです。 お客様視点とWEBデザインの知識を活かし、クライアント様、ユーザー様の課題解決に貢献できるWEBデザイナーを目指しています。

好きなことは食べること。パン・カレー・グルメバーガーの美味しいお店をリサーチして食べに行くのが楽しみです。

SKILLS

  • デザイン

    制作の目的や課題解決の方向性を擦り合わせた上で、ベンチマークしたサイトを参考にしながらデザインの制作ができます。
    ユーザーの動線を意識し、サイトを訪れた方がストレスを感じずに目的の情報に辿り着くように情報設計をすることを特に意識して制作しています。

    使用できるツール

    Figma（デザインで主に使用）/Illustrator/Photoshop

  • コーディング

    HTML・CSSを用いて、デザインを正しく再現します。また、JavaScriptで動きの表現をつけることができます。
    コードには規則性を持たせ、誰が見ても分かりやすく無駄のないコーディングを意識しています。

    使用できるツール

    Visual Studio Code

STRENGTHS

  • 01

    次行程を考慮した制作

    前職ではチームで動くことが多かったため、それぞれの業務を知ることや次工程の方が気持ちよくスムーズに業務が行えるよう努めてきました。
    WEBデザインにおいてもコーディングでの再現性を意識したデザインやツールの使い方、修正時に困らないコーディングを意識しています。

  • 02

    的確なヒアリング力

    相手の立場に立った的確なヒアリングが得意です。前職では20代から80代までの幅広い年齢の方と電話やメールで対応し、 「どのような言葉を使ったら相手からスムーズに情報を聞き出せるか」を常に意識してきました。
    この力は Web制作の際にも活かすことができると考えます。

  • 03

    困難な状況に強い

    性格はふんわり優しい雰囲気と言われることが多いですが、実は困難な状況でも折れない芯のある人間です。 途中で投げ出すことが嫌いで、最後まできっちり仕上げます。
    またわからないことに直面した際は、自分で調べて解決できるよう心がけています。

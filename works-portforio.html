ポートフォリオ

2025.02

自身の経歴や制作物をまとめたポートフォリオサイトを制作しました

  • 制作範囲

    企画 / 情報設計 / デザイン / コーディング

  • ターゲット

    Webデザイナーを募集している企業の採用担当者様

  • 目的

    就職活動のため、これまでの制作物やそれに対する思考、経歴や強みを伝える。

  • 情報設計

    採用担当者様の視点でまず見たいのは制作物だと考え、MVのすぐ下にWORKSを配置しました。トップページのWORKS一覧はモックアップを使い、制作物が一目見て印象に残るようにしています。 WORKSの各詳細ページの下部には次に見たいコンテンツを想定した遷移ボタンを配置し、ストレスなく作品を見ていただけるような設計にしました。ABOUTでは私の強みがWebデザイナーとして どのように活きてくるか具体的にイメージしていただけるように工夫しました。

  • デザイン

    主役である「作品」そのものが最も魅力的に見えるよう、全体をクリーンでミニマルなデザインにまとめました。特にトップページのメインビジュアルでは、白を基調に明るい光が差し込むような背景や画像を配置することで、デザイナーとしての明るい未来やポジティブな姿勢を表現しています。
    また、落ち着いたアースカラーやソフトな色合いを中心に選び、読みやすく柔らかさも感じられるフォントを採用することで、私の穏やかで誠実な人柄や丁寧な仕事ぶりが伝わるよう工夫しています。
    作品詳細ページは情報量が多いため心地よい余白を意識し、サイトを訪れた方がストレスなく、じっくり見ていただけるようにしました。

  • 制作期間

    企画 / 情報設計

    1週間

    デザイン

    2週間

    コーディング

    2週間

  • 使用ツール

    Figma / Visual Studio Code / Illustrator / Photoshop

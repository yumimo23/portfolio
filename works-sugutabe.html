野菜宅配サービス LP

2024.09

URL

デジタルハリウッドSTUDIOの課題として、架空のオーガニック野菜配送サービスのLPを制作しました

  • 制作範囲

    デザイン / コーディング

  • ターゲット

    成長期の子どもがいるママ。そこそこ食にお金をかけることができ、食にこだわる自分と「ていねいな暮らし」が好き。時間に余裕がない。

  • 目的

    食へのこだわりが強い層に響くような内容にし、お試し購入につなげる。

  • 情報設計

    「手軽にオーガニック野菜 家族が喜ぶ食卓を」をサイトコンセプトとして設定しました。忙しくも子どものために食にこだわりたいターゲットのために、 オーガニック野菜の特色やメリットが一目でわかるようなコンテンツを配置しています。またお客様の声では共感しやすい子持ちママの声を採用し野菜を使った料理の写真も載せることで、 実際に宅配サービスを使うとどのような食卓になるのか連想できる設計にしました。

  • デザイン

    全体的にナチュラルで明るく、オーガニック野菜の敷居を低く感じさせる親しみのあるデザインを目指しました。購入ボタンは緑黄色野菜をイメージした鮮やかなカラーを使用し強調しています。 テキストの量が多いため読み飛ばされないように読んでもらいたい部分はマーカーを引き、コンテンツごとのデザインを変えることで飽きさせないように工夫しました。

  • 制作期間

    企画 / 情報設計

    4週間

    デザイン

    3週間

    コーディング

    4週間

  • 使用ツール

    Figma / Visual Studio Code / Illustrator / Photoshop
